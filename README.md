# HawkLedger

See your money clearly.

## 📌 Overview
HawkLedger is a personal finance application designed to help users track expenses, analyze financial behavior, and gain visibility over their monthly balance.

## 🚀 Features (MVP)
- Manual transaction input
- CSV import
- Monthly balance overview
- Expense categorization

## 🧠 Problem
Bank statements are hard to interpret and often fragmented across multiple sources, making it difficult for users to understand their financial situation.

## 💡 Solution
HawkLedger consolidates financial data into a single view, allowing users to track, categorize, and analyze their transactions.

## 🛠 Tech Stack (initial)
- Backend: Python (FastAPI)
- Database: PostgreSQL
