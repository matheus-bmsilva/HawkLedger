# Certainties / Assumptions / Questions (CSD Matrix)

## ✅ Certainties
- Users want to understand where their money is going
- Bank statements are difficult to analyze directly
- Visualization helps users make better financial decisions
- Users often have multiple bank accounts

## 🤔 Assumptions
- Users are willing to upload financial data (CSV files)
- A standardized input format (CSV) will be acceptable
- Users prefer a simple interface over feature-rich complexity
- Categorizing expenses will provide meaningful insights
- Most users review their finances on a monthly cicle than daily
- Users expect transactions to be categorized automatically or with minimal effort

## ❓ Questions
- Do users prefer manual entry or file upload?
- Should multiple file formats (PDF, XLSX) be supported?
- How much automation do users expect in categorization and insights?
- Do users actually want spending suggestions?
- How much effort are users willing to put into tracking finances?
